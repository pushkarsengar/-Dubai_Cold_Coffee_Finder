# ☕ Dubai Cold Coffee Finder

A real-time Streamlit web application designed to locate, filter, and analyze the nearest cold coffee spots, cafes, and food trucks across major areas in Dubai using geospatial data.

---
## 🚀 Features

* **Interactive Map:** View all coffee spots with custom markers using Folium.
* **Radius Search:** Filter coffee spots dynamically based on distance (in km) using Geopy.
* **Multi-Tab Layout:** * 📍 `Nearby Spots` - Explore real-time recommendations with dynamic UI cards.
  * 📊 `Analytics` - Deep market analysis with interactive charts.
  * 🏆 `Leaderboard` - Top-rated coffee venues.
* **Smart Filters:** Search by name, spot type, minimum rating, and operational status (Open/Closed).
